# Data Structures in Python

A comprehensive project that implements a wide range of data structures in Python, organized in a clean and modular way with interactive demos.

---

## Features

- Clean and well-structured implementations
- Interactive demo files for each data structure
- Covers fundamental to advanced topics
- Suitable for learning, practice, and interviews

## Included Data Structures

### Linear
- Arrays (Static, Dynamic, 2D)
- Linked Lists (Singly, Doubly, Circular)
- Stacks
- Queues (Simple, Circular, Priority, Deque)

### Non-Linear
- Trees (Binary Tree, BST, Heap, Trie, Segment Tree, Fenwick Tree)
- Graphs (BFS, DFS, Dijkstra, MST, etc.)

### Hash-Based
- Hash Tables (Chaining, Open Addressing)
- HashMap and HashSet

### Advanced
- Disjoint Set (Union-Find)
- Skip List
- Suffix Array
- Sparse Table
- Treap
- Rope
- KD-Tree
- QuadTree
- Octree

### Specialized
- Bloom Filter
- LRU Cache
- Interval Tree
- Order Statistic Tree
- Priority Queue
